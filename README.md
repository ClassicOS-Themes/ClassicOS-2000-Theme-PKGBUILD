# ClassicOS 2000 PKGBUILD

A PKGBUILD for fellow Arch users to install the "ClassicOS2000" theme.

## Build Dependencies

- `sassc`
- `meson`

Install with `sudo pacman -S sassc meson`

## Instructions

1. Clone the git repository
2. `cd` into the repo
3. `makepkg`
4. Install the resultant package with `pacman -U`.
